# Claude Code ワークショップ素材

AI木曜会「Codeって書いてるけど、コードは書かない」ワークショップ用の練習素材です。

## ダウンロード方法

1. このページの緑色の「Code」ボタンをクリック
2. 「Download ZIP」をクリック
3. ダウンロードしたZIPをデスクトップに展開

gitは不要です。

## フォルダ構成

```
claude-code-workshop/
├── demo1_整理前/       ← デモ①：フォルダ整理（バラバラなファイル）
├── demo2_作業ログ/     ← デモ②：日報・週報の自動生成（5日分の作業ログ）
├── demo3_スキル/       ← デモ③：スキルで日報をコマンド1つで作成
├── 練習/              ← ハンズオン用（最初に触るフォルダ）
├── CLAUDE.md          ← ルールブックのサンプル
└── README.md          ← このファイル
```

## 各デモで使うもの

### デモ① フォルダを自動で整理する
`demo1_整理前/` を開いて、Claude Codeに「このフォルダを整理して」と指示します。
ファイル名がバラバラ、拡張子もバラバラ、分類もされていないファイルが入っています。

### デモ② 日報・週報を自動で作る
`demo2_作業ログ/` を開いて、Claude Codeに「今週の週報を作って」と指示します。
5日分の作業ログが入っていて、これを元に日報や週報を自動生成します。

### デモ③ スキルで日報をコマンド1つで作成
`demo3_スキル/` にはスキルファイル（`.claude/commands/daily-report.md`）が入っています。
`/daily-report` と打つだけで、デモ②と同じ日報作成が自動で実行されます。

### ハンズオン：実際に触ってみよう
`練習/` を開いて、Claude Codeに指示を出す練習をします。
