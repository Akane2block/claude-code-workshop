今日の日報を作成してください。

## 手順

1. `demo2_作業ログ/` フォルダの中から、一番新しい日付のファイルを見つける
2. そのファイルの内容を読む
3. 以下のフォーマットで日報を作成する
4. 作成した日報を `demo2_作業ログ/` フォルダに `日報_YYYYMMDD.md` という名前で保存する

## 日報フォーマット

```
# 日報 YYYY/MM/DD（曜日）

## 本日の実績
- （作業ログの「やったこと」を箇条書きで）

## 明日の予定
- （作業ログの「メモ」から次にやるべきことを抽出）

## 所感
- （作業内容を踏まえた一言コメント）
```

## ルール
- 文体はですます調
- 箇条書きは「- 」を使う
- 所感は1〜2文で簡潔に
